Description du site :

-YOUNIQUE- est un mini-site vitrine et interactif pour un salon de beauté 
situé en Algérie. Il propose la présentation et le filtrage dynamique des 
prestations de coiffure, de soins et d'onglerie, ainsi qu'un espace de simulation 
de réservation et de connexion utilisateur sans backend (via localStorage).

2. HAMOUR INES


Instructions d'utilisation :

1. Extraire l'archive du projet en respectant l'arborescence demandée.
2. Ouvrir le fichier "index.html" à la racine dans n'importe quel navigateur.
3. Pour tester l'authentification simulée :
   - Allez sur la page "Connexion".
   - Utilisez le compte de test suivant : 
     * Email : amel@gmail.com
     * Mot de passe : 123456
   - Ou créez un nouveau compte via l'onglet "Inscription" (les données sont sauvées dans le navigateur).
4. Pour le filtrage, allez sur la page "Prestations" et cliquez sur les boutons de catégories.
